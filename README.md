# BOOK LIBRARY API

## Exercise API NODEJS BACKEND
BECODE exercise : API creation with NODE JS of a book library

## Techs
- NodeJs
- ExpressJS : Routing
- MongoDB : Database
- Mongoose : Mongodb Object Modeling
- Template EJS
