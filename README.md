# BOOK LIBRARY API

## Exercise API NODEJS BACKEND
BECODE exercise : API creation with NODE JS of a book library with basic front.

## Techs
- NodeJs
- ExpressJS : Routing
- MongoDB : Database
- Mongoose : Mongodb Object Modeling
- Template EJS
- Heroku


# How to use?
After cloning, Run `NPM` to access the API on http://localhost:3000/
